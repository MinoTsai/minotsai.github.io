災難情境模擬演練

災難情境模擬演練，為高雄醫學大學-社工系、屏東科技大學-社工系、輔仁大學-社工系共同執行之計畫。本網站設計的主要目的為讓「社工學系」的同學模擬演練過去的災難事件發生時，社工於各個場域所面臨的人物及事件的真實狀況。

設計需求：因災難的發生是較為沈重的事情，因此在開始進行演練後，希望學生能融入整體較為抑鬱沈重的氛圍，因此在整體色調上轉為深色系。

訓練進行流程：「選擇災難事件」>「觀看災難前導影片」>「災難應變路徑圖（選擇地點及身份）」>「選擇事件腳本」>「聆聽及閱讀事件內容」>「小組討論」>「參考做法」>「簡易測驗及回饋」。

＊流程及內容持續討論及調整中，目前以「天然災害-台南地震」作為範本，可進行一次完整的模擬演練流程。

Github 原始碼 : https://github.com/MinoTsai/websit-design-DSSW
